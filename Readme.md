- Easy to use 
- Custom, Simple, Free, Active!, Updated!

## Features

- Categories
- Items
- Pictures
- Buy/Sell Prices
- Messages
- Colors of All Text
- Commands!
- Type of Amount Input (Slider, StepSlider, TextBox, Toggle)
- Enchants (Coming Soon..... "Can be Done Using Commands")
- Make Suggestions!
